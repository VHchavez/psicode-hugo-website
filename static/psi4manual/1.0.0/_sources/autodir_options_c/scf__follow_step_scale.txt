:term:`FOLLOW_STEP_SCALE <FOLLOW_STEP_SCALE (SCF)>`
"""""""""""""""""""""""""""""""""""""""""""""""""""

      When using STABILITY_ANALYSIS = FOLLOW, how much to scale the step along the eigenvector by. A full step of pi/2 corresponds to a value of 1.0. 

      * **Type**: double
      * **Default**: 0.5
