:term:`D_CONVERGENCE <D_CONVERGENCE (SCF)>`
"""""""""""""""""""""""""""""""""""""""""""

      Convergence criterion for SCF density, which is defined as the RMS value of the orbital gradient. See Table :ref:`SCF Convergence & Algorithm <table:conv_scf>` for default convergence criteria for different calculation types.

      * **Type**: :ref:`conv double <op_c_conv>`
      * **Default**: 1e-6
